This Example of Python code 

 
#!/usr/bin/python3

f = open('fout','w')
x = 0
while True:
  x += 1
  f.write(str(x))
  f.write('\n')

Explain :

#!/usr/bin/python3
This compiles file with compiler of python3

f = open('fout','w') This command open file by open() function and write the output in file fout
w   Stands for write in the file

x = 0 This a variable which has initial value = 0

while True : While is loop running until condition is satisfied
True : is the condition

x += 1 : This is counter which add 1 every time while loop iterate

f.write(str(x)) : Write the output in file fout as string
str() : is the function which return values as string

f.write("\n")
\n Make new line
